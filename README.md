# Clock CLI to Android App

This workspace now includes a basic Android Studio project that mirrors the main actions available from the CLI:

- connect to a clock device by IP
- fetch configuration
- set brightness
- update city name
- restart the device

## Next steps

1. Open the project in Android Studio.
2. Sync Gradle.
3. Run the app on an emulator or physical Android device.

## Notes

The Android app uses the same HTTP endpoints as the Python CLI and is intended as a starting point for a fuller mobile UI.
